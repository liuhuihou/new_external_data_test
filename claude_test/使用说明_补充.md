# 使用说明补充

## 启动方式

在项目目录执行：

```powershell
cd E:\code_file\ping_an_bank\new_external_data_test\claude_test
python server.py
```

浏览器打开：

```text
http://127.0.0.1:8765/
```

如果 8765 端口被占用：

```powershell
$env:PORT=8780
python server.py
```

## 页面操作

1. 上传 `CSV` 或 `XLSX`
2. 在页面里选择字段映射
3. 点击分析
4. 进入报告页生成智能报告
5. 在 API 配置区切换模型和请求地址

## 智谱配置

当前工作区已预置 `zhipu` profile，走 OpenAI 兼容接口：

- `api_type`: `openai`
- `api_url`: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
- `model`: `glm-5.2`

只需要在 `api_config.json` 或网页配置区里填入 API Key 即可。

## CLI

```powershell
python main.py serve
python main.py config show
python main.py upload duotou.xlsx
python main.py analyze duotou.xlsx --y-col flagy --bad-value 1 --query-flag-col flag_applyloanhourcust --variable-col alhc_h24_id_nbank_orgnum
python main.py report duotou.xlsx --html report.html
```

## 说明

外部智能 API 只接收聚合后的分析摘要，不会把原始明细整表直接发出去。
