# 外部数据测试工作台使用说明

## 1. 文件说明

- `server.py`：本地后端服务，负责上传、解析、分析、报告生成和外部 API 通信。
- `main.py`：命令行入口。
- `entrypoints/`：CLI、SDK、MCP 入口层。
- `agents/`：Agent 编排层，负责串联上传、分析、报告工具。
- `tools/`：工具层，封装上传、分析、报告能力。
- `services/api/`：API 通信边界。
- `api_config.json`：API 配置文件，用于填写请求地址、API Key、模型等信息。
- `外部数据测试SOP工作台.html`：前端页面。
- `duotou.xlsx`：示例文件。

## 2. 启动网页

```powershell
cd E:\code_file\ping_an_bank\new_external_data_test\claude_test
python main.py serve
```

浏览器访问：

```text
http://127.0.0.1:8765/
```

## 3. 前端配置

API 配置已拆到独立的「API 设置」页（`api_settings.html`），可从工作台右上角进入。左侧管理 Profile（新增 / 重命名 / 删除 / 切换生效），右侧编辑请求地址、API Key、模型等字段。

点击“保存配置”后会写回后端 `api_config.json`，并立即生效。详见 `docs/API配置说明.md`。

## 4. 命令行入口

### 启动服务

```powershell
python main.py serve
```

### 查看配置

```powershell
python main.py config show
python main.py config path
```

### 上传文件

```powershell
python main.py upload duotou.xlsx
python main.py upload duotou.xlsx --json
```

### 执行分析

```powershell
python main.py analyze duotou.xlsx --y-col flagy --bad-value 1 --query-flag-col flag_applyloanhourcust --variable-col alhc_h24_id_nbank_orgnum
```

### 生成报告

```powershell
python main.py report duotou.xlsx --markdown report.md --html report.html
```

## 5. SDK 用法

```python
from entrypoints.sdk import WorkbenchSDK

sdk = WorkbenchSDK()
result = sdk.analyze_path("duotou.xlsx")
print(result.data["analysis"]["grade"])
```

## 6. MCP 入口

```powershell
python main.py mcp
```

支持的工具：

- `upload_dataset`
- `analyze_dataset`
- `generate_report`

## 7. 数据流

前端或 CLI -> `server.py` -> `agents/DataTestingAgent` -> `tools/` -> 本地分析 / 外部 API 报告生成。

外部 API 只接收聚合后的分析摘要，不发送原始明细行。

## 8. 常见问题

### 页面打不开

先确认后端已启动，再访问 `http://127.0.0.1:8765/`。

### 上传后分析失败

检查文件是否为 CSV 或 XLSX，且首行是字段名。

### 报告显示本地生成

说明外部 API 未配置、不可用，或调用失败，系统已自动回退到本地规则报告。
